个人简介

我正在学习 Python、MySQL、数据分析和 Web 开发，希望从单纯学习语法，逐步转向能够独立完成真实项目。

这个门户网站是我的第一个完整 Web 项目，它会记录我的学习过程、项目进度、技术笔记和阶段成果。

当前学习方向

当前重点方向包括：Python 编程、MySQL 数据库、HTML/CSS 网页制作、GitHub 使用、网站部署，以及后续的 Flask 后端开发。

我的目标不是只会看懂代码，而是能够真正完成一个从本地开发到公网部署的完整流程。

为什么要做这个网站？

只刷题容易停留在局部知识点，例如某个语法、某个函数、某道题的解法。

但做网站需要把多个知识点组合起来：页面结构、页面样式、文件管理、版本控制、部署上线、后端逻辑和数据库。

所以这个网站不只是一个展示页面，也是我训练工程能力的起点。

技术路线

第一阶段：HTML + CSS + GitHub Pages，完成静态门户网站。

第二阶段：Python + Flask + MySQL，完成动态网站和后台管理。

第三阶段：Linux 服务器 + Nginx + Gunicorn + 域名，完成真正的公网部署。

我的阶段目标

短期目标：完成一个可以被公网访问的个人门户网站。

中期目标：做出带数据库、登录、文章管理功能的网站。

长期目标：具备独立开发、部署、维护小型 Web 项目的能力。
