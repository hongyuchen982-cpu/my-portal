学习笔记

这里记录我的 Python、MySQL、Web 开发和项目学习过程。

一、Python 学习记录

当前学习重点：Python 基础语法、函数、文件操作、异常处理、面向对象，以及后续的数据分析方向。

当前目标不是死记语法，而是能用 Python 解决实际问题，例如读取文件、清洗数据、处理表格、自动化重复操作。

二、MySQL 学习记录

当前学习重点：数据库、数据表、增删改查、主键、外键、索引、视图、函数、存储过程和触发器。

MySQL 的核心价值是存储和管理数据。以后做真正的网站时，用户信息、文章内容、项目数据都可以存进数据库。

三、Web 开发学习记录

当前阶段：先学习 HTML 和 CSS，理解网页是如何被浏览器显示出来的。

HTML 负责网页结构，例如标题、段落、链接、图片；CSS 负责网页样式，例如颜色、字体、间距和布局。

下一阶段计划学习 Flask，用 Python 写后端，让网站可以连接 MySQL，实现文章发布、登录、后台管理等功能。

四、刷题与算法记录

刷题的目的不是为了背答案，而是训练逻辑思维。当前重点可以放在数组、字符串、哈希表、双指针、栈、队列、树结构这些基础内容。

但刷题不能替代项目。刷题训练局部能力，项目训练完整系统能力。两者都要有，但当前更应该尽快做出一个能上线的网站。

五、阶段目标

第一阶段目标：完成一个静态门户网站，包括首页、项目展示、学习笔记和关于我页面，并成功部署到 GitHub Pages。

第二阶段目标：使用 Flask 和 MySQL 重构网站，让网站从“展示页面”升级为“可以管理内容的动态网站”。
